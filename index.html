อาหาร

Description

ถ้าระบบหมุนเวียนเลือดดี สุขภาพร่างกายก็จะดีไปด้วย และหากตอนนี้รู้สึกเลือดไหลเวียนไม่สะดวก อยากช่วยกระตุ้นการไหลเวียนของโลหิตตัวเอง ลองรับประทานอาหารเพื่อสุขภาพเหล่านี้ เพื่อช่วยในไหลเวียนโลหิต [กดปุ่มข้างล่างอ่านเพิ่มเติม]

อ่านเพิ่มเติม

ออกกำลังกาย

Description

การออกกำลังกายอย่างน้อยวันละ 30 นาที สัปดาห์ละ 4- 5 ครั้ง ช่วยลดความเสี่ยงต่อโรคหัวใจได้ ทั้งยังลดความเสี่ยงต่อโรคอื่น ๆ รวมทั้งโรคร้ายแรงอย่างเช่น โรคมะเร็ง การออกกำลังกายช่วยกระตุ้นการไหลเวียนโลหิต ทำให้หัวใจแข็งแรง ทำงานอย่างมีประสิทธิภาพ ช่วยลดน้ำหนักส่วนเกิน พักผ่อนคลายความเครียด ทำให้อารมณ์ดี และกระดูกแข็งแรง
[กดปุ่มข้างล่างเพื่อดูข้อมูลตารางความเหมาะสมของความหนักการออกกำลังกาย]

ดูข้อมูลเพิ่มเติม

สุขภาพจิต

Description

คงไม่อาจปฏิเสธได้ว่าความเครียดเป็นส่วนหนึ่งของชีวิต แต่หากเครียดมากเกินไปก็ส่งผลร้ายต่อสุขภาพกาย อารมณ์ และสุขภาพจิตความเครียดมีผลทำให้ความดันโลหิตเพิ่มขึ้น ส่งผลให้เกิดแนวโน้มที่จะมีพฤติกรรมทำลายสุขภาพ เราจึงควรเสริมสร้างความเครียดโดยใช้ “ ความคิดเชิงบวกต่อสุขภาพ ” ในการดูแลความเครียดของเรา

แนวทางการสร้างเสริม


สุขภาพจิตด้วยความคิดเชิงบวกการสร้างเสริมสุขภาพจิตด้วยความคิดเชิงบวก มีเทคนิคคล้ายคลึงกับเทคนิคการสร้างความสุขด้วยตนเอง โดยมีพื้นฐานจากแนวคิดทางจิตวิทยาเชิงบวก ประกอบด้วย การพัฒนาจุดแข็งของบุคคล และปรับวิธีการคิดให้เป็นบวกร่วมกับการดูแลตนเองด้านสุขภาพกาย
ข้อที่ 1

คบหากับเพื่อนมีความคิดบวก จะช่วยให้เราเกิดความรู้สึกดีๆ และมีการแลกเปลี่ยนความคิดด้านดี

ข้อที่ 2

ฝึกทำสมาธิเป็นการทำให้จิตใจเข้มแข็ง มีสติ สามารถเข้าใจตนเองและปรับปรุงตนเองได้เสมอ

ข้อที่ 3

คิดในมุมกลับ หลักการคือไม่คิดแต่เพียงมุมมองของตัวเอง จะทำให้ลดความโกรธ สามารถเข้าใจผู้อื่นได้มากขึ้น

ข้อที่ 4

คิดหลายมุม ด้วยการมองให้รอบด้าน วิเคราะห์ให้เห็นถึงสาเหตุปัจจัยต่าง ๆ ทำให้ได้คำตอบที่หลากหลายแง่มุมมากขึ้น

ข้อที่ 5

การยิ้มและหัวเราะเป็นสิ่งที่สะท้อนอารมณ์ได้ดีว่ามีความสุขหรือความทุกข์ และยังเป็นการปรับอารมณ์ด้านลบให้ดีขึ้นได้

ยาเสพติด

Description

การติดยาเสพติดในกลุ่มเด็กและเยาวชน มักจะเนื่องมาจากครอบครัว ไม่เป็นสุขมีความขัดแย้ง หรือปล่อยปะละเลยจนถูกชักจูงไปเสพได้ง่าย หรือใช้ยาเสพติดเป็นทางออกของชีวิต จะทำให้เกิดโรคทางระบบไหลเวียนโลหิตเฉียบพลัน เช่น หัวใจวาย หรือ โรคหลอดเลือดสมอง

วิธีการป้องกันให้ห่างจากยาเสพติด

ข้อที่ 1

สร้างบรรยากาศในครอบครัวให้มีความสุข จะทำให้เด็ก มีความมั่นคงทางจิตใจ รู้สึกอบอุ่น

ข้อที่ 2

อธิบายให้เด็กเข้าใจและรู้โทษตามกฎหมายที่จะได้รับจากการใช้ยาเสพติดให้เด็กได้รู้โทษของยาเสพติด

ข้อที่ 3

จัดกิจกรรมที่เสริมสร้างพลังใจ อารมณ์ ความนึกคิดไปในทางที่มีความหมาย เช่น ชมรมกีฬา ชมรมดนตรี

ข้อที่ 4

ร่วมมือกับครูประจำชั้นเพื่อหาทางแก้ไข เช่นจัดกลุ่มอภิปรายกับเด็กนักเรียน ถึงพิษภัยและโทษของยาเสพติด

ข้อที่ 5

พาเด็กไปพบแพทย์ตามสถานที่รับรักษาผู้ติดยาเสพติด หรือสถานที่บางแห่งรักษาด้วยสมุนไพร

ข้อที่ 5


รู้จักบริหารจัดการกับความวิตกกังวลและการพักผ่อนที่เพียงพอ

การพักผ่อนมีความสําคัญมากกับอวัยวะทุกส่วนของร่างกายคนสมองต้องการหยุดตัดสินใจ การพักผ่อนของหัวใจใช้เวลาพักผ่อนประมาณ 6-8 ชั่วโมง

ข้อที่ 6


การตรวจเช็คร่างกายตามระยะเวลาที่เหมาะสมการตรวจสุขภาพอย่างสม่ำเสมอ


ตรวจสุขภาพอย่างสม่ำเสมอตามระยะเวลาที่เหมาะสม จะช่วยลดอัตราการเสียชีวิตจากโรคร้ายบางโรคได้

สมุนไพร

Description

สมุนไพรเป็นสิ่งที่คนในสมัยปัจจุบันไม่ได้ให้ความสนใจมาก แต่สรรพคุณขอสมุนไพรต่างๆก็สามารถนำมารักษาหรือเสริมสร้างทำให้ระบบไหลเวียนโลหิตของเราไหลเวียนได้ ไม่เกิดโรคได้ง่ายๆ สมุนไพรที่ช่วยเสริมสร้างระบบไหลเวียนโลหิต
